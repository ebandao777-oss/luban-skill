# baseline-skill

## 概述

一个已知质量的轻量参考 skill，供 luban-skill 消费者能力基线测试（Phase 3）使用。该 skill 功能明确、结构完整、无外部依赖，作为"已知好"的基线参照物。

## 触发条件

当用户询问以下问题时使用此技能：
- "列出当前目录的文件"
- "显示文件列表"
- "查看目录内容"
- "dir"
- "ls"

## 执行流程

### Phase 1：确认目标目录

读取用户指定的目录路径。若用户未指定，默认使用当前工作目录。

异常处理：
- 目录不存在 → 提示用户并请求重新指定
- 路径是文件而非目录 → 列出该文件的父目录

### Phase 2：列出文件

使用 `shell_executor` 执行 `Get-ChildItem` 列出目录内容。

输出格式：
```
[文件名](<绝对路径>)
```

### Phase 3：汇总展示

按文件类型分组展示（文档/图片/代码/其他），每组内按名称排序。

## 约束

1. 禁止列出系统目录（C:\Windows, C:\Program Files 等）
2. 隐藏文件和系统文件默认不展示（除非用户明确要求）
3. 结果超过 50 个文件时分页展示
